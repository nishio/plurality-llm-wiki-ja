# クリエイティブなコラボレーション

本書第 5-3 章 [[5-3 クリエイティブなコラボレーション]] のタイトルにもなっている、本書 [[プルラリティ]] の中核実践分野の 1 つ。英語 creative collaboration。

## 本書での扱い

[[5-0 協働テクノロジーと民主主義]] が示す [[プルラリティ]] の協働モード一覧の 1 つとして、[[ポスト表象コミュニケーション]]、[[没入型共有現実]]、[[拡張熟議]]、[[適応型管理行政]]、[[多元投票]]、[[社会市場]] と並んで配置される ([[社会的差異を超えたコラボレーションのための技術]] 参照)。

[[社会的差異を超えたコラボレーションのための技術]] の文脈で、[[鈴木健]] が [[オードリー・タン]] に [[プルラリティ]] における collaboration の意味合いを尋ねた回答にも、共同創造としての collaboration が中心に据えられる ([[社会的差異を超えたコラボレーションのための技術]] 参照)：

> 「私は『コラボレーション』という言葉を用いて『共創』や『ピアプロダクション』を表現し、共有された創造的なプロセスを強調しました」 ([[plurality-book]] [[日本語版解説]] より、[[社会的差異を超えたコラボレーションのための技術]] 引用)

## 含意

- ピアプロダクション (Wikipedia 的協働、オープンソース開発)
- [[GFM]] を介した人間と AI の共同創作
- 本書自身の執筆プロセスで使われている [[⿻管理プロトコル]]
- [[クアドラティック資金調達]] による創作プロジェクトへの資金調達

## 本書での位置づけ

[[クリエイティブなコラボレーション]] は本書 [[プルラリティ]] が掲げる「[[社会的差異を超えたコラボレーションのための技術]]」の射程を、政治的熟議 ([[拡張熟議]]) や経済 ([[社会市場]]) だけでなく、文化・創作領域にまで拡張する位置づけとなる。

## Updates

### 2026-05-27: 日本側で自発的に発生した creative collaboration の事例

本書 [[5-3 クリエイティブなコラボレーション]] は本書執筆プロセス自体 (CC0 GitHub での集合執筆、[[⿻管理プロトコル]]) を中核例として論じる。日本コミュニティ側でも本書の精神を体現する creative collaboration が、刊行前から自発的に発生していた。

- ヒップホップ・グループ [[Dos Monos]] が 2024 年頃に発表した『Civil Rap Song ft. Audrey Tang 唐鳳』── [[オードリー・タン]] が音声サンプリングで参加し、Yuma Kishi が AI Art 映像を制作、Kei Wakabayashi (blkswn publishers) がプロデュース。学術書や講演ではなく**ヒップホップという大衆芸術ジャンル**での Plurality 表現で、本書のいう「[[ポスト表象コミュニケーション|ポスト表象]]」が文化生産でどう機能するかの稀少事例 ([[Dos Monos]] / [[plurality-japanese-scrapbox]]より)
- [[plurality-japanese-scrapbox]] そのもの (1421 page) が、本書 [[5-3 クリエイティブなコラボレーション]] の論じる「ピアプロダクション」の日本独自版。50 人以上の参加者による訳語論争・講演翻訳・概念解説の集合知形成プロセスは、本書本体の GitHub workflow とは別の文体・別のツール (Scrapbox の `[link]` 記法と `.icon` 発言者明示) で並列実装されている
- 本書本文の [[⿻管理プロトコル]] (コミュニティ通貨とグループ ID トークンによる creative collaboration の resource allocation) と並列的に、日本側では [[Funding the Commons Tokyo]] 2024 が [[クアドラティック資金調達]] を実演する場となっている ([[Funding the Commons Tokyo]]より)
