# ⿻管理プロトコル

英語名 Plural Management Protocol。本書 [[plurality-book]] 自身の執筆・運営・貢献者へのインセンティブ設計に使われている、複数管理プロトコル。

## 本書での扱い

[[plurality-book]] ページの解説によれば：

> GitHub 上でのイシュー / プルリクエストによる修正提案や、コミュニティ通貨とグループ ID トークンを用いた ⿻管理プロトコル ([[5-3 クリエイティブなコラボレーション]]、[[5-7 社会市場]] 参照) など、書籍そのものが [[プルラリティ]] を実践している ([[plurality-book]] 引用)

[[5-7 社会市場]]「明日の社会市場」節の中で、複数管理の実装として位置づけられる ([[社会市場]] 参照)：

> [[⿻管理プロトコル|⿻マネジメント]]: 本書作成で使用された複数管理プロトコル ([[plurality-book]] [[5-7 社会市場]] より、[[社会市場]] 引用)

## 仕組み

- GitHub 上でのイシュー / プルリクエストによる修正提案
- コミュニティ通貨 + グループ ID トークンによる貢献者・グループへのインセンティブ配分
- [[クアドラティック資金調達]] と類似する逓減比例ルールで「重複した利益」と「多様な支持」を区別

## 本書の自己言及性

本書は「書籍そのものが [[プルラリティ]] を実践する」自己言及的な構造を意図的に持っている。[[クアドラティック資金調達]] による Gitcoin 上の寄付獲得 ([[クアドラティック資金調達]] 参照) と並んで、⿻管理プロトコルはそのもう 1 つの柱となる。
