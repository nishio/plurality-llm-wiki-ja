# Audrey Tang Keynote in Plurality Tokyo

[[plurality-japanese-scrapbox]] 内の同名ページに保管されている、2023-04-14 [[Plurality Tokyo]] (#1) における [[オードリー・タン]] Keynote の英日対訳。

- 動画: [https://www.youtube.com/watch?v=JyAJrRFRCiA](https://www.youtube.com/watch?v=JyAJrRFRCiA)
- 翻訳: [[nishio]] (動画キャプションの英語を元に和訳、訳注付き)

## 主要主張

### 開会の詩 (mission を凝縮)

> When we see "Internet of Things", Let's make it an Internet of Beings.
> When we see "virtual reality", Let's make it a shared reality.
> When we see "machine learning", Let's make it collaborative learning.
> When we see "user experience", Let's make it about human experience.
> And whenever we hear that a singularity is near, let's always remember the plurality is here.

(モノのインターネット → 存在のインターネット、仮想現実 → [[没入型共有現実|共有現実]]、機械学習 → 協調学習、UX → ヒューマンエクスペリエンス、シンギュラリティ → プルラリティ)

[[シンギュラリティ]] (singular) と [[プルラリティ]] (plural) を対置する立論の最も簡潔な定式化として頻繁に引用される。

### Plurality の定義の言い換え

「独裁国家 がテクノロジーを支配の手段として利用するのに対し、私たちはテクノロジーを民主化し、プルーラリティあるいは [[協調的多様性]] (collaborative diversity) を目指しています」

[[nishio]] の訳注: 「独裁」の「単一」である性質に対して「プルーラリティ」を対置している (「独裁国家」は本書の英訳プロセスでこの対置を強調する訳語選択)。

### 玉山 (Mount Jade) への言及

「ブロードバンドを人権とし、すべての国民がたとえ台湾の最高峰である [[玉山からの眺め|玉山]] ででもライブストリーミングに十分な帯域幅を確保できるようにし」(本書の [[玉山からの眺め]] と接続するエピソード)。

### [[総統杯ハッカソン|台湾総統杯ハッカソン]] と [[クアドラティック投票|QV]]

「台湾総統が主催するハッカソンにおける [[クアドラティック投票|Quadratic Voting]] などの規範的メカニズムにより、多くの共創イニシアチブが実現されています」── Water Refill Map、CO2 吸収プラットフォーム、河川管理プラットフォーム LASS など。

### MODA (デジタル省) の役割

「台湾のデジタルトランスフォーメーションの『MODA』または『モーター』として、私たちは市民とテクノロジーの橋渡しをし、産業とセキュリティを一体化させ強化します」(2022-08 設立)。

### 日本の Web3 イベント群への共感

「東京で展開される Web3 イベントの数々、例えば「[[Plurality Tokyo]]」「ETH Global Tokyo」「DAO Tokyo」などを目の当たりにして、胸が熱くなった」── 分散型台帳を通じて Plurality を具現化するとの位置づけ。

## このソースの価値

[[plurality-book]] (書籍) より前に行われた Plurality の口頭での定式化として、本書を補完する独立 source。特に「IoT → Internet of Beings」「VR → shared reality」「ML → collaborative learning」「UX → human experience」のフレーズ群は、書籍にはこの形で出現せず、Tang の対話的・詩的なメッセージング戦略を観察できる。
