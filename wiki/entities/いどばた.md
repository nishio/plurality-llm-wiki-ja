# いどばた

[[デジタル民主主義 2030]] が開発する大規模熟議プラットフォーム。

## 機能

> 台湾の JOIN や [[vTaiwan]] を参考にした大規模熟議プラットフォーム。AI を使って、議論すべき課題の抽出、解決策の検討、解決策の評価と合意形成を支援する ([[plurality-in-japan-overview]] より、dd2030.org/idobata/)

[[ブロードリスニング]] が「聞く」技術だとすると、いどばたは「**議論し、整理し、合意形成へつなぐ**」技術として位置づけられる。

## 本書での扱い

本書本文・[[日本語版解説]] には登場しない。Plurality 日本語版刊行 (2025) 以降に開発が進行している、本書 [[5-4 拡張熟議]] の日本独自実装の一つ。

## Updates

### 2026-05-28: 2 モジュール構成と運用拡張

GitHub README (`github.com/digitaldemocracy2030/idobata`, GPL-3.0) で 2 モジュール構成が articulate されている:

- **いどばたビジョン**: テーマに対する「未来像」や「方向性」を広く集め、論点を整理
- **いどばた政策**: 具体的な施策案を改善・ブラッシュアップ (GitHub プルリクエスト経由)

[[team-mirai-manifesto]] 10 章では「**マニフェストに対する提案を AI が引き出し、実際にわずか 3 日で 1000 件以上の意見を収集**」「対面イベント (みらいいどばた会議) でも、数分で参加者全員の本音を引き出し可視化」という具体実績が報告。「政党・自治体・企業・学校など多様な組織が利用・ホスティング可能な**公共財**として提供」と GitHub README で位置づけられている。

### 2026-05-28: [[拡張熟議]] 3 レベル framework での位置

[[team-mirai-manifesto]] のデジタル民主主義 3 レベル framework において:

- レベル 1 (整理・可視化): [[広聴AI]] / Talk to the City
- レベル 2 (引き出す): **いどばた** + AI インタビュー (本ツールが該当)
- レベル 3 (合意形成): **いどばたシステム拡張** (論点投票 → 重要論点議論 → 具体提案議論の 3 段階)

参照: vTaiwan / Habermas Machine (Google DeepMind) / 京大 D-Agree システム ([[team-mirai-manifesto]] 10 章より)。
