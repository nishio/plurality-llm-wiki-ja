# Code for Japan

[https://www.code4japan.org/](https://www.code4japan.org/)

> 「ともに考え、ともにつくる」をモットーに、市民主体で社会課題を解決していくコミュニティ。

代表: 関治之 (hal sk)。

## Plurality 文脈での役割 ([[plurality-japanese-scrapbox]]より)

- 2023-04-14 [[Plurality Tokyo]] (#1) の主要 supporter (RadicalxChange Foundation と並ぶ)
- [[オードリー・タン]] Keynote の日本語字幕付き動画を Code for Japan YouTube チャンネルで公開
- 関治之 (hal sk) が Software Design 2023 年 7 月号に [[第139回 民主主義を発展させるためのテクノロジー「Plurality」]] を寄稿

## シビックテック文脈

関治之 (hal sk) は「シビックテックの活動を始める前は『ちゃんと納税していれば、あとは行政の仕事でしょ』と思っていた」が、「Code for Japan の活動で各地を回ることで、あらためて民主主義の大切さを実感することになった」と書いている ([[第139回 民主主義を発展させるためのテクノロジー「Plurality」]]より)。

彼にとって [[プルラリティ]] は「市民が主体的に公共に関わり、自分たちの住む地域を改善していくシビックテック」の発想を、テクノロジーで本格的に発展させる可能性を持つもの ([[第139回 民主主義を発展させるためのテクノロジー「Plurality」]]より)。

## Updates

### 2026-05-27: 本書 [[7-0 政策]] / [[2-2 デジタル民主主義の日常]] の日本側 institutional 対応物

本書 [[2-2 デジタル民主主義の日常]] が描く台湾の [[g0v]] / [[vTaiwan]] エコシステムに対応する日本側 institutional 主体としての Code for Japan の位置づけは、post-book でより明示化されている。

- [[第139回 民主主義を発展させるためのテクノロジー「Plurality」]] (関治之, Software Design 2023-07) は本書 [[plurality-book]] 日本語版刊行の 2 年前に、日本の技術者コミュニティに向けて [[プルラリティ]] を初期紹介した寄稿。「シビックテックの実践者から見た Plurality」という独自視点は本書本文に登場しない pre-book context として、本書 [[7-0 政策]] の主張の日本ローカル受容を準備した
- [[安野貴博]] 都知事選 → [[デジタル民主主義 2030]] への lineage で、Code for Japan は実装 partner / institutional anchor として継続的に機能している。本書 [[5-4 拡張熟議]] 末尾の都知事選事例は、Code for Japan が 10 年以上育ててきたシビックテック基盤の上に成立している

### 2026-05-27: [[DigDAO]] との Quadratic Funding 実験

2024-07 に [[DigDAO]] と共同で、シビックテック・デジタル公共財への資金分配に [[クアドラティック資金調達]] を使う実験を発表 ([[plurality-in-japan-overview]]より、PR TIMES プレスリリース経由)。従来型公共調達・助成金に乗りにくいプロジェクトに対し、ユーザーの支持をもとに民主的に資金を分配する仕組み。本書 [[5-7 社会市場]] 議論の日本シビックテック領域への応用。
