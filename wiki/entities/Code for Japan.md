# Code for Japan

[https://www.code4japan.org/](https://www.code4japan.org/)

> 「ともに考え、ともにつくる」をモットーに、市民主体で社会課題を解決していくコミュニティ。

代表: 関治之 (hal sk)。

## Plurality 文脈での役割 ([[plurality-japanese-scrapbox]]より)

- 2023-04-14 [[Plurality Tokyo]] (#1) の主要 supporter (RadicalxChange Foundation と並ぶ)
- [[オードリー・タン]] Keynote の日本語字幕付き動画を Code for Japan YouTube チャンネルで公開
- 関治之 (hal sk) が Software Design 2023 年 7 月号に [[第139回 民主主義を発展させるためのテクノロジー「Plurality」]] を寄稿
- 2023-05-13 の Audrey Tang 真鶴視察にも関与

## シビックテック文脈

関治之 (hal sk) は「シビックテックの活動を始める前は『ちゃんと納税していれば、あとは行政の仕事でしょ』と思っていた」が、「Code for Japan の活動で各地を回ることで、あらためて民主主義の大切さを実感することになった」と書いている ([[第139回 民主主義を発展させるためのテクノロジー「Plurality」]]より)。

彼にとって [[プルラリティ]] は「市民が主体的に公共に関わり、自分たちの住む地域を改善していくシビックテック」の発想を、テクノロジーで本格的に発展させる可能性を持つもの ([[第139回 民主主義を発展させるためのテクノロジー「Plurality」]]より)。
