# バラジ・スリニバサン

英語名 Balaji Srinivasan (1980-)。インド系アメリカ人の起業家・投資家・著述家。元 Andreessen Horowitz (a16z) ジェネラルパートナー、Coinbase の元 CTO。

## 本書での扱い

[[2-0 IT と民主主義 拡大する溝]]「暗号資産とハイパー資本主義」節で、[[リバタリアニズム]] の「明確な知的規範とリーダーの集団」を構成する人物の 1 人として言及される：

> ジェームズ・デール・デイビッドソンとウィリアム・リース＝モッグ卿による『独立主権の個人』、メンシウス・モルドバグというペンネームでカーティス・ヤービンが著した著作、バラジ・スリニバサンによる『ネットワーク国家』、および『青銅器時代のマインドセット』（いずれも未邦訳）は、このコミュニティで広く読まれ、引用されている ([[plurality-book]] [[2-0 IT と民主主義 拡大する溝]] より)

参考文献として Balaji Srinivasan, *The Network State* (Self-published, 2022, https://thenetworkstate.com/) が挙がる。

## 人物

『The Network State: How To Start a New Country』(2022) で、暗号通貨と分散型ガバナンスを基盤に、地理的領域に縛られない「ネットワーク国家」を立ち上げるロードマップを提示。クリプト・リバタリアン陣営における代表的論者の 1 人。

## 本書での位置づけ

[[リバタリアニズム]] の現代における代表的論者として短く言及される。本書は『ネットワーク国家』の主張に立ち入った論評は行わず、リバタリアン陣営の知的体系の存在を読者に示すために言及する。
